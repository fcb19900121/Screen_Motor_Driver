# Motor Driver USB 虚拟串口通讯协议

## 1. 物理接口

| 项目 | 说明 |
|------|------|
| 接口类型 | USB 2.0 Full Speed, CDC ACM (虚拟串口) |
| 驱动 | Windows 自动识别为 COM 口, 无需额外驱动 |
| 波特率 | 无实际意义 (USB CDC), 串口工具可设任意值 |
| 数据格式 | ASCII 文本 |
| 行结尾 | `\r\n` (CR+LF) |
| 最大命令长度 | 63 字节 (不含行结尾) |

## 2. 命令格式

```
命令名:参数\r\n
```

- 所有命令均为 **大写 ASCII**
- 命令以 `\r\n` 结尾 (也可仅 `\n`)
- 设备收到完整命令后立即返回响应

## 3. 命令列表

### 3.1 俯仰电机控制 — `PITCH`

控制 Motor2 (直线推杆) 运动到指定位置。

**格式:**

```
PITCH:<百分比>\r\n
```

**参数:**

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| 百分比 | 整数 | 0 ~ 100 | 行程百分比, 0=零点, 100=最大行程 |

**内部映射:** 0~100 → 编码器计数 0~40960

**示例:**

| 发送 | 含义 |
|------|------|
| `PITCH:0\r\n` | 运动到零点位置 |
| `PITCH:50\r\n` | 运动到 50% 行程 (编码器 20480) |
| `PITCH:100\r\n` | 运动到最大行程 (编码器 40960) |

**前提条件:** 俯仰电机必须完成上电回零 (Homing), 否则返回错误。

---

### 3.2 屏幕旋转控制 — `SCREEN`

控制 Motor1 进行横屏/竖屏切换。

**格式:**

```
SCREEN:PORTR\r\n    (切换到竖屏)
SCREEN:LANDS\r\n    (切换到横屏)
```

**参数:**

| 参数 | 说明 |
|------|------|
| `PORTR` | Portrait, 竖屏方向 |
| `LANDS` | Landscape, 横屏方向 |

**示例:**

| 发送 | 含义 |
|------|------|
| `SCREEN:PORTR\r\n` | 旋转到竖屏位置 (LIMIT1 限位) |
| `SCREEN:LANDS\r\n` | 旋转到横屏位置 (LIMIT2 限位) |

**说明:** 到达对应限位开关后自动停止。若已在目标位置则不动作。

---

### 3.3 重新回零 — `RESET`

触发俯仰电机重新回零 (Homing) 及屏幕方向重新探测。

**格式:**

```
RESET\r\n
```

**无参数。**

**示例:**

| 发送 | 含义 |
|------|------|
| `RESET\r\n` | 俯仰电机重新回零 + 屏幕方向重新探测 |

**说明:** 执行后俯仰电机进入 Homing 状态, PITCH 命令在回零完成前返回 ERR:NOT_READY。可用于 FAULT 恢复或手动重新校准。

---

## 4. 响应格式

设备收到命令后返回以下响应之一:

| 响应 | 含义 |
|------|------|
| `OK\r\n` | 命令已接受并开始执行 |
| `ERR:NOT_READY\r\n` | 俯仰电机未完成回零, 无法执行 PITCH 命令 |
| `ERR:BAD_VALUE\r\n` | PITCH 参数格式错误或无法解析 |
| `ERR:UNKNOWN\r\n` | 未识别的命令 |

> **注意:** `OK` 表示命令已被接受, 不代表运动已完成。运动为异步执行。

## 5. 通讯示例

```
PC → 设备:  PITCH:50\r\n
设备 → PC:  OK\r\n

PC → 设备:  SCREEN:LANDS\r\n
设备 → PC:  OK\r\n

PC → 设备:  PITCH:abc\r\n
设备 → PC:  ERR:BAD_VALUE\r\n

PC → 设备:  HELLO\r\n
设备 → PC:  ERR:UNKNOWN\r\n
```

## 6. 系统行为说明

| 行为 | 说明 |
|------|------|
| 上电回零 | Motor2 上电自动执行 Homing, 完成前 PITCH 命令返回 ERR:NOT_READY |
| 屏幕自检 | 上电自动探测屏幕当前方向 (Flash 记录或往返探测), 自检期间 SCREEN 命令可正常接受 |
| 掉电保护 | 屏幕方向保存在 Flash, 旋转前写入 MOVING_TO 状态, 到位后写入最终状态 |
| 角度保护 | 屏幕旋转超过 ±95° 软件限位自动停止 |
| 重复命令 | 已在目标位置时, SCREEN 命令不会重复动作; PITCH 会重新执行运动 |
